При помощи localStorage можно хранить пользовательские данные, которые нам нужны даже если пользователь выключил комп, закрыл браузер. localStorage хранится на жёстком диске, так что пока винду не переустановить, данные будут доступны.

Объём localStorage ограничен 10-20 Мб в зависимости от браузера. Если попытаться записать что-нибудь в уже заполненный localStorage, то браузер выкинет ошибку, поэтому при работе с localStorage можно использовать try-catch
